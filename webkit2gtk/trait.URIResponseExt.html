fn get_content_length(&self) -> u64

Get the expected content length of the URIResponse. It can be 0 if the server provided an incorrect or missing Content-Length.

Returns

the expected content length of self.

fn get_mime_type(&self) -> Option<GString>

Returns

the MIME type of the URIResponse

fn get_status_code(&self) -> u32

Get the status code of the URIResponse as returned by the server. It will normally be a soup::KnownStatusCode, for example soup::Status::Ok, though the server can respond with any unsigned integer.

Returns

the status code of self

fn get_suggested_filename(&self) -> Option<GString>

Get the suggested filename for self, as specified by the 'Content-Disposition' HTTP header, or None if it's not present.

Returns

the suggested filename or None if the 'Content-Disposition' HTTP header is not present.
